Linux系统中配置和启动Redis服务

发表于 2023-11-22 更新于 2023-11-23 分类于 notes

服务启动

方法一：执行/usr/local/redis-5.0.9/src/目录中的redis-server
cd到/usr/local/redis-5.0.9/src/，即你安装redis的目录下的src文件夹，执行./redis-server

方法二：启动服务并加载配置文件（推荐，可以后台运行redis）
cd到/usr/local/redis-5.0.9/，执行 src/redis-server ./redis.conf

停止服务

在服务启动界面control+C

连接

  1. 无密码连接：
    新建一个终端窗口，cd到/usr/local/redis-5.0.9/src/目录，执行./redis-cli
    出现127.0.0.1:6379> 说明已经连接上本地6379

  2. 带密码连接：cd到/usr/local/redis-5.0.9/，src/redis-cli -h localhost -p 6379 -a 你的密码

退出连接

localhost:6379>exit

设置后台运行

先停止服务，修改/usr/local/redis-5.0.9/redis.conf配置文件。修改过程具体如下：

  • cd /usr/local/redis-5.0.9/
  • vim redis.conf
  • /dae 回车 查找关键词
  • i，将daemonize 的no改为yes，配置redis服务启动后在后台运行
  • 再启动一次，但是需要显式加载哪个配置文件：
    • cd /usr/local/redis-5.0.9
    • 执行src/redis-server ./redis.conf

希望客户端连接redis服务时需要提供密码

在Linux服务器上安装MySQL

发表于 2023-11-22 更新于 2023-11-23 分类于 notes

首先检测当前系统中是否安装MySQL数据库。
rpm -qa 查询当前系统中安装的所有软件。qa意思是query all
rpm -qa | grep mysql 查询当前系统中安装的名称带mysql的软件
rpm -qa | grep mariadb 查询当前系统中安装的名称带mariadb的软件

RPM (Red-Hat Package Manager)RPM软件包管理器，是红帽Linux用于管理和安装软件的工具。
注：如果安装过了mariadb，需要先卸载。否则mysql会安装失败。
卸载软件命令：rpm -e --nodeps 软件名称。例如rpm -e --nodeps mariadb-libs-xxxxxxx

安装和配置MySQL的步骤

  1. 下载mysql8.0.31 arm64版本（适用于macOS m2芯片）。mysql-8.0.31-linux-glibc2.17-aarch64.tar.gz
    下载地址：https://downloads.mysql.com/archives/community/

  2. 将下载好的mySQL安装包上传到linux并解压

  • 在本地终端，用sftp传输该安装包
    put /Users/kathy/Downloads/linux-software/mysql-8.0.31-linux-glibc2.17-aarch64.tar.gz /root/

  • 在虚拟机终端，解压安装包到/usr/local/mysql(先用mkdir mysql创建好)
    tar -zxvf mysql-8.0.31-linux-glibc2.17-aarch64.tar.gz -C /usr/local/mysql

  • 在虚拟机终端，在/usr/local/mysql-8.0.31-linux-glibc2.17-aarch64/下，创建包data和log

  1. 修改环境变量：
    cd ~

vim .bash_profile

其中添加：PATH=$PATH:$HOME/bin:/usr/local/mysql-8.0.31-linux-glibc2.17-aarch64/bin

修改完后source .bash_profile

which mysql - 会显示/usr/local/mysql-8.0.31-linux-glibc2.17-aarch64/bin/mysql

在Linux服务器上部署项目

发表于 2023-11-21 更新于 2023-11-22 分类于 notes

手工部署项目

  1. 打包：在IDEA中开发一个Springboot项目，并打成jar包

IDEA右侧maven，点击package即可打包。打包为target/helloworld-0.0.1-SNAPSHOT.jar。

  1. 将jar包上传到linux服务器
  • mkdir /usr/local/app 创建目录，准备将项目jar包放在此目录
  • 打开终端，点击菜单中的shell，选择新建远程连接
  • 选择安全文件传输sftp，点击右边服务器下的加号，添加你要连接的远程服务器ip地址（例如192.168.50.158）
  • 选中右边的服务器，在“用户”一栏输入服务器对应的用户名（例如root），下方出现“sftp root@192.168.50.158”，点击连接。
  • 输入密码，连接成功
  • 输入put 本地文件路径 远程主机路径命令，即put /Users/kathy/Desktop/workspace_idea/helloworld/target/helloworld-0.0.1-SNAPSHOT.jar /usr/local/app/，上传文件到/usr/local/app/。
  1. 启动springboot程序

在虚拟机终端执行 java -jar helloworld-0.0.1-SNAPSHOT.jar

1.1 MySQL主从复制

1.1.1 介绍

MySQL主从复制是一个异步的复制过程，底层是基于Mysql数据库自带的二进制日志功能。就是一台或多台MySQL数据库(slave，即从库）从另一台MySQL数据库(master，即主库）进行日志的复制然后再解析日志并应用到自身，最终实现从库的数据和主库的数据保持一致。MySQL主从复制是MySQL数据库自带功能，无需借助第三方工具。

MySQL复制过程分成三步：

  • master将改变记录到二进制日志（binary log)
  • slave将master的binary log拷贝到它的中继日志（relay log）
  • slave重做中继日志中的事件，将改变应用到自己的数据库中

1.1.2 配置-前置条件
